# Introduction to Aomi

Aomi is agentic AI infrastructure for onchain applications. It connects AI models to your APIs and blockchain tools, letting users research, query, and execute transactions through a natural language interface — on web or Telegram.

## Key Concepts

Two concepts are central to how Aomi works — worth understanding before you dive in:

**[Namespace](/docs/namespace)** — The isolated agent environment that defines what the AI can do and access. Both users and builders interact within a namespace; switching namespaces changes the agent's tools and context entirely.

**[Non-Custodial Wallet](/docs/non-custodial-wallet)** — Aomi never holds your private keys or funds. Every transaction is simulated on a forked network before it reaches your wallet, so you can review the exact on-chain outcome before approving.
