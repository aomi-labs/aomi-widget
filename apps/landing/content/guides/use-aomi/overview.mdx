# Use Aomi

Aomi lets you ask questions, run actions, and complete transactions from natural-language chat. This track is for end users and operators using Aomi in product UIs or Telegram.

## What You Can Do

- Ask questions and get streamed responses in real time
- Trigger tool-backed actions (quotes, balances, market lookups)
- Connect a wallet and approve transactions when required
- Continue conversations across sessions and channels

## Where You Can Use It

### Web UI

Use Aomi inside a product website/app with a full chat interface.

### Telegram

Use Aomi through a Telegram bot with slash commands, panels, and wallet support.

See [Telegram Overview](/docs/use-aomi/telegram/overview).

## Typical Transaction Flow

1. Ask for an action, for example a quote or trade.
2. Aomi gathers context and calls the required tools.
3. If signing is required, review the wallet request.
4. Approve or reject.
5. Receive confirmation and continue the conversation.

## Safety Basics

- Always verify destination addresses and amounts before signing.
- Reject any request you do not understand.
- Use the correct chain/network before approval.
