# About Aomi Widget

Aomi Widget helps you create beautiful AI-powered chat interfaces with wallet integration built-in.

## Key Features

### AI-Powered Chat

Built on top of `@assistant-ui/react`, Aomi Widget provides a complete chat interface with support for:

- Streaming responses
- Message history
- File attachments
- Tool calling

### Wallet Integration

Seamlessly integrated with Reown AppKit for:

- Multi-chain support
- Wallet connection
- Transaction signing
- Account management

### Customization

Fully customizable with:

- Theme variants (light/dark modes)
- Custom styling
- Component composition
- Runtime configuration
