# MetaMask

This is an upcoming demo showcasing how Aomi chat can be embedded directly within wallets like MetaMask. The goal is to bring agentic capabilities — tool calling, streaming responses, and on-chain operations — into the wallet interface itself, so users can interact with Aomi Apps without leaving their wallet.
