# MetaMask

Build a MetaMask-style wallet interface powered by the Aomi AI assistant. These endpoints
cover account queries, transaction submission, message signing, and chain switching —
all forwarded to the connected wallet for user approval.

All endpoints require an `X-Session-Id` header. Write operations (`sendTransaction`,
`signMessage`, `switchChain`) return wallet-approval requests.
