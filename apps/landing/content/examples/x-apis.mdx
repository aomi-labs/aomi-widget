# X APIs

Aomi integrates the **X (Twitter) API v2** for social intelligence —
look up user profiles, read timelines, search posts with advanced operators,
and track trending topics. All endpoints require a Bearer token from the
[X Developer Portal](https://developer.x.com). Aomi wraps these as tools
so your AI assistant can query social data and cross-reference it with
on-chain activity.
