# Examples

These examples demonstrate how Aomi integrates with real-world platforms to build agentic applications. Each platform featured here offers public, free APIs — which makes them ideal candidates for showcasing the full integration pipeline: wrapping endpoints as AI tools, configuring an Aomi App with a preamble and model, and deploying it as a live app.

The purpose is not to endorse any specific platform, but to show how any product with accessible APIs can become an Aomi-powered agentic application. The patterns here — tool design, preamble structure, app configuration — apply directly to your own integration.

## Platform Applications

These are production integrations built on public APIs:

- [Polymarket](/examples/polymarket) — prediction market data and on-chain trading via Polymarket's public API
- [DeFi Aggregators](/examples/defi-aggregators) — cross-chain swaps, liquidity, and yield across DeFi protocols
- [X APIs](/examples/x-apis) — social data and sentiment as AI-accessible context
