# Polymarket

Polymarket exposes three public API surfaces — **Gamma** for market metadata,
**Data API** for trade history, and **CLOB** for order book and order submission.
Aomi wraps these as tools so your AI assistant can search markets, resolve
user intent, and execute trades through the connected wallet.
