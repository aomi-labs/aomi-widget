# DeFi Aggregators

Aomi integrates DeFi data and trading through multiple API surfaces —
**DefiLlama** for market intelligence (prices, yields, TVL, bridges),
**0x** and **LI.FI** for on-chain swap execution, and **CoW Protocol** for
intent-based batch settlement. Aomi wraps each of these as tools so your AI
assistant can look up prices, compare routes, and execute trades through the
connected wallet.
