# Components

Add components via shadcn presets, then import from `@aomi-labs/react`.

<WidgetDemo />

## Core exports

- `AomiFrame` — full assistant shell with chat surface and thread list
- `AomiChat` — chat-only experience when you want to control layout yourself
- Wallet sidebar surfaces — add RPC + session config before enabling

## Usage

Use `npx shadcn add` with the provided preset URLs for frame or wallet-only installs. Import components from `@aomi-labs/react`, wrap your app with the provided providers, and include `@aomi-labs/react/styles.css` for tokens.
