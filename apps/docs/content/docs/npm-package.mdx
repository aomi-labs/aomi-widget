# npm Package

Install the widget from npm to embed it in your app.

```bash
pnpm add @aomi-labs/react
```

## Peer dependencies

Include the listed peer deps (assistant-ui, Radix primitives, Tailwind utilities) to match the versions used in this repo.

## Updates

Follow semver; pin versions in apps that ship to customers and refresh after verifying in the docs site.
