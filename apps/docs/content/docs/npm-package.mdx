# NPM Package

Install the widget and pull in the shadcn preset:

```bash
npm install @aomi-labs/react
npx shadcn add @aomi-labs
```

For full preset variants:

```bash
# Full install (frame)
npx shadcn add https://your-domain.com/r/aomi-frame.json

# Just the wallet sidebar
npx shadcn add https://your-domain.com/r/aomi-wallet-sidebar.json
```

Include the listed peer deps (assistant-ui, Radix primitives, Tailwind utilities) to match the versions used in this repo. Follow semver; pin versions in apps that ship to customers and refresh after verifying in the docs site.
