# Welcome to Aomi Widget

Aomi Widget is an AI-powered assistant UI widget library built with React and Next.js.

## Features

- Pre-built AI chat components
- Wallet integration with Reown AppKit
- Customizable themes
- Server-sent events for real-time updates
- TypeScript support

## Quick Start

Get started with Aomi Widget in your Next.js project:

```bash
npm install @aomi-labs/react
```
